# kulilimambu · Next.js

## 로컬에서 보기

```bash
npm install
npm run dev
```

그 다음 **Chrome 또는 Safari**를 열고, 주소창에 **반드시 아래만** 입력하세요:

```
http://localhost:3000
```

- **Cursor/IDE 미리보기 사용하지 마세요.** 404가 나올 수 있습니다.
- **일반 브라우저**에서 **http://localhost:3000** 을 직접 입력해야 메인 화면이 나옵니다.

포트 3000이 이미 쓰이면 터미널에 나온 다른 번호(예: 3001)로 접속하면 됩니다.
