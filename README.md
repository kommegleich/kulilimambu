# kulilimambu 포트폴리오

## ⚡ 바로 보기 (서버 없음, 즉시)

1. **`바로보기.command`** 파일을 **더블클릭**하세요.  
   → 브라우저에서 포트폴리오가 바로 열립니다.
2. 또는 **`index.html`** 파일을 더블클릭해도 됩니다.

인터넷이 연결되어 있어야 예시 이미지/동영상이 보입니다.

---

## 📁 Next.js 버전 (site 폴더)

배포용·개발용으로 Next.js 버전을 쓰려면:

1. 터미널에서:
   ```bash
   cd site
   npm install
   npm run dev
   ```
2. **Chrome 또는 Safari**를 열고, 주소창에 **그냥** 아래만 입력해서 접속:
   ```
   http://localhost:3000
   ```
3. **Cursor 미리보기나 IDE 내장 브라우저 말고**, **일반 브라우저**에서 위 주소를 직접 입력해야 합니다.
